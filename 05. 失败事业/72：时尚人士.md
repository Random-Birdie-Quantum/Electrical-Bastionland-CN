没什么比那些缺少现代外貌的人更可悲了

**特殊**：如果你在着装时没有发出声明，你陷入窘困。

**范例名字**：感受，注释，波动，模式

若你是最年轻的玩家
团队欠下了£10K的债务，债主是……
**狮子议会**：

| 你获得                 |
| ------------------------ |

| 在抉择许久之后，你最终选中了什么不实用的武器？            |
| ------------------------ |
| **£1** **一对双钩**（单独使用时d4，成对使用时d6）。 |
