但它们都不值得我们崇拜

| ------------------------------------------------------------ |

| **2HP** 你只是**热爱人类**，并想要保持它的纯净。获得火焰弹（在熄灭之前每轮造成d6伤害）。 |

| **5HP** 大部分是为了**找乐子**。获得一桶啤酒。 |  
| **6HP** 你的家在一次**不明的爆炸**中被摧毁。获得烧伤的皮肤。 |
